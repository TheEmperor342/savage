How to use this API

Here is a list of all the images currently available


A basic example of this api can be:

https://svgimageapi.herokuapp.com/api?img=python



That's cool, but how do I change its images?


Its fairly easy too. Here's an example:

https://svgimageapi.herokuapp.com/api?img=python&_1=fab387&_2=f9e2af&_3=94e2d5&_4=74c7ec


To add colors to the image you have to append parameters like _1, _2, _3, _4
and so on to the request URL with the colors in hex format.
